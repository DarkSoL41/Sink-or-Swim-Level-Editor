Format (reference)

Header: "MAP." + width(u16) + height(u16) + 120 reserved bytes.
Then: width×height cells, 2 bytes each (LE).
Background tile = index into a LINEAR array of 32×32px blocks, 1024 bytes each (confirmed via disassembly: offset = id×0x400). Level 1 uses the BLOCKL1A+B+C bank (0-179).
